# kinship_terms_rsl
Социолингвистический анализ вариативности в терминах родства в РЖЯ 

Данная работа посвящена анализу социолингвистической вариативности терминов родства в РЖЯ. Для анализа были выбраны жесты, обозначающие такие термины, как: мама, папа, родители, сестра, брат, дочь, сын, бабушка, дедушка, тетя, дядя, внучка, внук.
Исследование направлено на выявление значимых связей между вариантами жестов и социолингвистическими факторами (регион рождения, возраст, пол, уровень владения РЖЯ). 

## Requirment
- pandas==3.0.0
- networkx==3.6.1
- numpy==2.4.1
- matplotlib==3.10.0
- seaborn==0.13.2
- statsmodels==0.14.6

## Основные используемые библиотеки 
- pandas - чтение 'csv' файла и работа с таблицей.
- networkx - создание вершин и ребер, поиск циклов и связанных вершин.
- numpy - работа с массивом данных.
- matplotlib и seaborn - построение графиков. 
- statsmodels - использование статистических тестов.
